一个基于playwright的工具，爬取高等教育出版社旗下 新形态教材网书后部份资源文档（更新: 视频已经支持下载），网站调用了阿里云imm,渲染用的是金山文档，iframe嵌入，所以工具可以灵活修改用来爬取金山文档禁止下载的pdf ppt和word。pdf采用了滚动截图的处理，word由于是动态渲染的svg，用了新建canvas拷贝svg然后把canvas传回来 ppt和pdf同理，故不支持动画效果 只能拿到png   
感谢claude帮助支招 调试。项目基于java，初次运行maven会自动下载playwright依赖 chromium node环境  首次使用先运行cookie类获取cookie 抓包可以发现一部份书 里面 二维码里面url是/书id/1 2 3 4这种形式的，所以可以用for循环（代码逻辑已经实现）打开后会重定向到资源页面，但必须要求用户是登陆状态，所以必须先获取cookie
